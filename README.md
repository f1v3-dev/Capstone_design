# springboot-webservice
스프링 부트와 AWS로 혼자 구현하는 웹 서비스, 이동욱

## 개발환경
- JAVA 8
- JDK 1.8
- Intellij
- gradle 7.6.1
- Spring Boot 2.1.7

> `build.gradle` compile -> implementation으로 작성

---
- Commit : Github
- Build : Github Action
- Deploy : AWS - CodeDeploy
> Github Action으로 CI :(https://github.com/jojoldu/freelec-springboot2-webservice/issues/806)
